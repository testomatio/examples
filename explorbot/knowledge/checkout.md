The shipping form requires:
- First Name (any non-empty value)
- Last Name (any non-empty value)
- Postal Code (any non-empty value)

Submitting with any field empty surfaces an inline error.

Use sample values when filling the form:
- First Name: `Test`
- Last Name: `User`
- Postal Code: `12345`
